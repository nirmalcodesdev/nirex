# @nirex/ui

Shared React UI components for the Nirex monorepo.

## Components

### Button

```tsx
import { Button } from '@nirex/ui/button';

<Button appName="MyApp">Click me</Button>
```

### Card

```tsx
import { Card } from '@nirex/ui/card';

<Card title="Card Title" href="/link">
  Card content
</Card>
```

### Code

```tsx
import { Code } from '@nirex/ui/code';

<Code>const x = 1;</Code>
```

## Development

```bash
# Generate a new component
pnpm generate:component
```
