# @nirex/shared

Shared utilities and types for the Nirex monorepo.

## Contents

- **Constants** - Application-wide constants
- **Types/Interfaces** - Shared TypeScript definitions

## Usage

```typescript
import { APP_NAME, API_VERSION, StandardResponse } from '@nirex/shared';
```

## Exports

| Export | Type | Description |
|--------|------|-------------|
| `APP_NAME` | `string` | Application name |
| `API_VERSION` | `string` | API version |
| `StandardResponse<T>` | `interface` | Standard API response shape |
