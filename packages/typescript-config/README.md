# @nirex/typescript-config

Shared TypeScript configurations for the Nirex monorepo.

## Configurations

### base.json

Base TypeScript configuration for Node.js packages.

```json
{
  "extends": "@nirex/typescript-config/base.json"
}
```

### react-library.json

TypeScript configuration for React libraries.

```json
{
  "extends": "@nirex/typescript-config/react-library.json"
}
```

## Usage

Add as a dev dependency:

```bash
pnpm add -D @nirex/typescript-config
```
